### RL_CPong

```
Dependences:
- raylib 5

Installation with Docker:
- docker run -it --rm -v "$PWD":/usr/src/app -w /usr/src/app ubuntu:22.04 sh -c "./configure.sh && ./build.sh"

Build with Docker:
- docker run -it --rm -v "$PWD":/usr/src/app -w /usr/src/app ubuntu:22.04 bash
- ./configure.sh
- ./build.sh
```
